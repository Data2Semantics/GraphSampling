The html files linked to from this page allows you to analyze our sampled subgraphs more in-depth. We serve an html page for every dataset. Each html pages shows the recall for each query, and for each of the generated subgraphs. For the DBPedia results, the SP2Bench results and the Linked Movie Database results, each query is clickable and executable as well. Other datasets do not have this features, as we cannot distribute the queries for these datasets
